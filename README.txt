Как запустить онлайн-версию (GitHub Pages)
--------------------------------------------
1) Загрузите файлы `index.html`, `style.css`, `game.js` в репозиторий на GitHub.
2) В Settings → Pages выберите: Source: Deploy from a branch, Branch: main / (root). Сохраните.
3) Ссылка будет вида https://ваш_логин.github.io/имя_репозитория

Изменения в этой версии:
- Исправлены клики по кнопкам на iOS (включены обработчики touchstart + z-index).
- Экран-приветствие «Привет, Радмир! ...» показывается до старта.
- Финиш — тортик, сообщение «С днём рождения!».
